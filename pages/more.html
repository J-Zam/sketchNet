Core Features of Sketch-Net

What is Sketch-Net?

Sketch-net is a doodle classifier based on a large dataset of doodles using a Convolutional Neural Network (CNN) called DoodleNet which is trained with the quickdraw-dataset of Google (a collection of 50 million drawings).

Thanks to ml5.js , a JavaScript library built on TensorFlow.js that simplifies machine learning, making it accessible for creative projects, data analysis, and interactive experiences. It abstracts the complexity of machine learning, allowing easy integration into web applications.

Finally, p5.js for creating an interactive drawing canvas and handling user input.

What is a Convolutional Neural Network?

A Convolutional Neural Network (CNN) is a type of deep learning model commonly used for processing data that has a grid-like structure, such as images. CNNs are especially effective for tasks like image classification, object detection, and facial recognition.

The main layers of a Convolutional Neural Network (CNN) are the Convolutional Layer, Activation Layer (ReLU), Pooling Layer, Fully Connected Layer, and Output Layer.

How CNNs interpret images?

when you draw on the canvas, the CNN interprets the pixels by applying a series of convolutional filters to detect features, then applies pooling to reduce dimensionality, and finally uses fully connected layers to classify the doodle. The CNN has learned to recognize important patterns in the pixel data, and by feeding it the right preprocessed image, it can predict the content of the doodle with high accuracy. The better the CNN is trained on diverse doodles, the more accurate it will be in interpreting new drawings.
